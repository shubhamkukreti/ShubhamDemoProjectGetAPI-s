Steps to start the project

1- Set the MySQL path & username, password in "application.properties" file accordingly. 
2- Run the "ProductOrderSystemApplication.java" file.
3- Open the "ProductOrderSystemApplicationTests" file and execute "insertProduct" function only.
4- Open the  "Insert_Data_In_Table.sql" file and execute the queries.
5- You are ready to perform the API requests.
